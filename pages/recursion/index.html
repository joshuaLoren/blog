Recursion

Recursion occurs when you call a method within itself. Lets look problem often used to explain the basics of recursion: the factorial problem. 


                        import java.io.*;
                        import java.util.*;
                        
                        public class Solution {
                        
                            public static void main(String[] args) {
                                System.out.println(GCD(1000000,999994));
                            }
                            
                            /**
                             *@param x and y - two integers
                             *@return - GCD
                             */
                            public static int GCD(int x, int y){
                                
                                int temp;
                                //place in correct order, x needs to be greater than y
                                if(y > x)
                                {
                                    temp = y;
                                    y = x;
                                    x = temp;
                                    System.out.println(x);
                                    System.out.println(y);
                                }
                                //base case
                                if(y == 0)
                                {
                                    return x;
                                }
                                else 
                                {
                                    return GCD(y, (x % y));
                                }
                            }
                            
                        }

                                if(y == 0)

                                }
